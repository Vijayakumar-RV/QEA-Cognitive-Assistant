### US-09-01 — Valid Login Credentials
**Type:** Manual  |  **Priority:** High  
**Tags:** Login, My Account

**Description**  
Verify that the login form accepts valid credentials and redirects to My Account dashboard

**Preconditions**
- User has a valid account and is on the login page

**Test Steps**
1. Enter valid username and password in the login form
2. Click the Login button

**Expected Result**
> User is redirected to the My Account dashboard

---
### US-09-02 — Invalid Login Credentials
**Type:** Manual  |  **Priority:** High  
**Tags:** Login, Error Message

**Description**  
Verify that the login form shows an error message for invalid credentials

**Preconditions**
- User is on the login page

**Test Steps**
1. Enter invalid username and password in the login form
2. Click the Login button

**Expected Result**
> Error message is displayed indicating invalid credentials
