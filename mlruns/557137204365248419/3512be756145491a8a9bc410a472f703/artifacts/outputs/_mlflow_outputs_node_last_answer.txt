### US02-TC001 — Verify Search Bar is present on the Home Page
**Type:** Manual  |  **Priority:** High  
**Tags:** search, UI, home_page

**Description**  
To ensure the search bar is visible and accessible on the Home Page as per the user story.

**Preconditions**
- User is on the Home Page.

**Test Steps**
1. Navigate to the application URL: https://www.automationteststore.com/

**Expected Result**
> The 'Search Bar' input field is visible on the Home Page.

### US02-TC002 — Search for a product using keywords from the Home Page
**Type:** Manual  |  **Priority:** High  
**Tags:** search, functionality, home_page

**Description**  
To verify that a user can enter a keyword into the search bar and initiate a search.

**Preconditions**
- User is on the Home Page.

**Test Steps**
1. Navigate to the application URL: https://www.automationteststore.com/
2. Enter a valid product keyword (e.g., 'Makeup') into the 'Search Bar'.
3. Press Enter to submit the search.

**Expected Result**
> The system navigates to the 'search_results_page'.

### US02-TC003 — Search for a non-existent product from the Home Page
**Type:** Manual  |  **Priority:** Medium  
**Tags:** search, negative, home_page

**Description**

**Preconditions**

**Test Steps**

2. Enter a non-existent product keyword (e.g., 'NonExistentProduct123') into the 'Search Bar'.
3. Press Enter to submit the search.

**Expected Result**
> The system navigates to the 'search_results_page'.
