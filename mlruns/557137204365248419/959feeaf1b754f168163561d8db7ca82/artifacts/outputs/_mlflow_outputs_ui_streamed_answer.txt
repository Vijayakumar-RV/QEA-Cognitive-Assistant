### US-09-01 — Valid Login Credentials
**Type:** Manual  |  **Priority:** High  
**Tags:** login, account, authenticated

**Description**  
Verify that the login form accepts valid credentials and redirects to My Account dashboard

**Preconditions**
- User has a valid account and is on the login page

**Test Steps**
1. Enter valid login name in the 'Login Name' input field
2. Enter valid password in the 'Password' input field
3. Click the 'Login Button'
4. Verify that the user is redirected to the My Account dashboard page

**Expected Result**
> User is successfully logged in and redirected to the My Account dashboard page

---
### US-09-02 — Invalid Login Credentials
**Type:** Manual  |  **Priority:** High  
**Tags:** login, error, invalid_credentials

**Description**  
Verify that the login form shows an error message for invalid credentials

**Preconditions**
- User is on the login page

**Test Steps**
1. Enter invalid login name in the 'Login Name' input field
2. Enter invalid password in the 'Password' input field
3. Click the 'Login Button'
4. Verify that an error message is displayed

**Expected Result**
> Error message is displayed for invalid login credentials
