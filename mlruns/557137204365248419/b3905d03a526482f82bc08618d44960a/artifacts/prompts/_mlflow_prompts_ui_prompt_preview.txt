### TC_US_02_02 — Search for existing products using valid keyword
**Type:** Manual  |  **Priority:** High  
**Tags:** search, functional, us-02

**Description**

**Preconditions**
- User is on the Home Page with the Search Bar visible.

**Test Steps**
1. In the Header, locate the Search Bar (Locator: name='search').
